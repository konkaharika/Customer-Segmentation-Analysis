# Customer Segmentation Dashboard

## Project Overview
This project analyzes customer demographics and spending behavior using Power BI.

## Tools Used
- Power BI
- CSV Dataset

## Features
- Customer Segmentation
- KPI Cards
- Scatter Plot Analysis
- Interactive Slicers
- Gender Analysis

## Dashboard Insights
- Potential customers form the largest segment.
- High Value customers have the highest spending scores.
- Customer segmentation helps identify target groups.
- Income and spending behavior are related.
